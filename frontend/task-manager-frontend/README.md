# Task Manager Frontend

Angular frontend for the Task Manager assignment.

## Scope

- Authentication screens (Login/Register)
- Task list with search and status filtering
- Create task and update task flows
- JWT-based protected routes via interceptor and guard

## Run Frontend Only

From this folder:

```bash
npm install
npm start
```

App URL: `http://localhost:4200`

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```
